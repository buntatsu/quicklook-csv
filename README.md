A QuickLook Plugin for CSV Files (日本語対応版)
===============================================

[QuickLookCSV][github_org]のShift_JIS、EUC-JP、JIS対応版。


インストール
------------

- **QuickLookCSV.qlgenerator**を `~/Library/QuickLook/` か `/Library/QuickLook/` にコピー。
- ターミナルで `qlmanage -r` を実行(もしくは再起動)。
